O git e o GitHub são apps que tem como objetivo guardar repositórios ou projetos sem que o usuário perca eles, em outras palavras ele é um OneDrive para programadores. 

Para guardar um repositório existente, vai em 'File->New Repository',e daí tem que preencher o nome que tem que igual ao da pasta, a descrição do repositório que tem que vai do programador, o endereço do documento a ser guardado e a licença. Depois disso tudo nós vamos poder publicar para a conta do site principal quer seja de forma pública ou privada.

O GitHub permite que mudemos os projetos pois ele informa que se mudou, dai informamos que atualizamos no summary e daí comitamos para que tire a mensagem e atualize, porém, ele não vai atualizar para o site principal, e para fazermos isso temos que fazer um push origin, aí sim ele vai atualizar no site também.

OBS: Para apagar um repositório no site e só ir em settings rolar até ao fim e ir em delete. O mesmo acontece se quisermos tirar ele do privado para o publico ou vice-versa.
